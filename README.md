# Africa Financial Tyranny Index

An index tracking financial repression against African civil society. Documenting how states abuse financial infrastructure to suppress dissent through arbitrary account freezes, foreign funding restrictions, mandatory state banking, and AML/CFT regulations weaponised for political control.

---

> ⚠️ **Project Status: Work in Progress**
> HakiFedha is a young project, and this index is an evolving work in progress. It is not exhaustive; it documents strictly confirmed, sourced incidents. The true scale of financial repression across Africa is certainly larger. Many incidents go unreported because organisations fear drawing further attention to themselves, while others lack coverage by credible media or legal documentation. 
> 
> Contributions, corrections, and new incident reports are welcome and encouraged at [hakifedha.org](https://hakifedha.org) or securely via email at `hakifedha@proton.me`.
